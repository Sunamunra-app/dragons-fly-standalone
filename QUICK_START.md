# 🎮 Quick Start - Dragon's Fly v11.0

## ⚡ Быстрый запуск (1 минута)

### 1. Локальный тест:

```bash
cd /home/user/dragons-fly-standalone
python3 -m http.server 8000
```

Откройте: **http://localhost:8000**

### 2. Проверьте консоль (F12):

Должны появиться:
```
✅ Managers created
✅ Game patched successfully!
✅ SDK initialized
🎵 Audio initialized (после клика "Играть")
```

### 3. Протестируйте функции:

- **Звук:** Кликните "Играть" → должна заиграть музыка
- **Пауза:** Смените вкладку → музыка остановится
- **Возобновление:** Вернитесь → музыка возобновится
- **Level Up:** Соберите 50 яиц → появится уведомление
- **Рекорд:** Проиграйте → рекорд сохранится

## 🐛 Быстрое решение проблем

### Звуки не воспроизводятся?
→ Кликните на кнопку "Играть" (Web Audio требует взаимодействия)

### Музыка не останавливается при смене вкладки?
→ Проверьте консоль на ошибки, убедитесь что подключен audioManager.js

### Уведомления о level up не показываются?
→ Соберите 50+ яиц, проверьте что подключен gamePatch.js

### Canvas обрезается на мобильных?
→ Используйте viewport-fit=cover (уже добавлено в index.html)
