## ✨ Особенности

- 🎨 **Пиксель-арт графика** в стиле 16-bit
- 🎯 **Динамичный геймплей** с нарастающей сложностью
- 🔥 **Система огненных шаров** - стреляй по препятствиям!
- 🍎 **Яблоки-ускорители** - восстанавливай заряд
- 🌍 **Смена дня и ночи** каждые 2 минуты
- ❤️ **3 жизни** - хардкорный режим
- 🏆 **Сохранение рекордов** (VK Storage)
- 📱 **Мобильная оптимизация**

---

## 🕹️ Управление

### ПК:
- **Стрелки / WASD** - управление драконом
- **Пробел** - выстрел огненным шаром
- **Мышь** - альтернативное управление

### Мобильные:
- **Тап по экрану** - управление и стрельба

---

## 🎯 Геймплей

1. **Избегай препятствий** - облака, камни, стрелы
2. **Собирай яйца** - каждое яйцо = +10 очков
3. **Стреляй файерболами** - уничтожай камни и препятствия
4. **Собирай яблоки** - восстанавливай заряд огня
5. **Выживай** - с каждой минутой сложность растёт!

## 🚀 Локальный запуск

```bash
# Скачайте репозиторий
git clone https://github.com/Sunamunra-app/dragons-fly-standalone.git
cd dragons-fly-standalone

# Запустите локальный сервер
python3 -m http.server 8000

# Откройте в браузере
http://localhost:8000
```

## 🔧 Интеграция с VK Mini App

### Шаг 1: Укажите URL в VK
```
1. Откройте https://vk.com/apps?act=manage
2. Создайте приложение "Dragon's Fly"
3. В поле "Адрес iFrame" укажите:
   https://sunamunra-app.github.io/dragons-fly-standalone/
```

### Шаг 2: Настройте права
```
Включите в настройках VK приложения:
✅ storage - для сохранения рекордов
✅ ads - для показа рекламы
```

### Шаг 3: Протестируйте
```
Откройте приложение в VK и проверьте:
✅ Игра загружается
✅ Рекорды сохраняются
✅ Реклама показывается каждые 3 игры
```
