## 🎮 Как это работает

### Инициализация:

1. **Загрузка страницы** → Подключаются все модули
2. **DOMContentLoaded** → `gamePatch.js` расширяет класс Game
3. **Инициализация SDK** → Автоопределение языка, загрузка рекордов
4. **Клик на "Играть"** → Инициализация Web Audio API
5. **Загрузка спрайтов** → Отправка Game Ready в Яндекс.Игры

### Во время игры:

- При сборе яйца → Обновление статистики, проверка достижений
- При повышении уровня → Уведомление + награды
- При получении достижения → Уведомление + монеты
- При смене вкладки → Автопауза музыки
- При сворачивании → Автопауза музыки

### Game Over:

- Остановка музыки
- Сохранение рекорда (localStorage + SDK)
- Обновление статистики
- Показ рекламы (каждые 3 игры)

## 🐛 Решение проблем

### Проблема: Звуки не воспроизводятся

**Решение:**
- Web Audio API требует взаимодействия пользователя
- Убедитесь что кликнули на "Играть" перед началом игры
- Проверьте консоль на ошибки

### Проблема: Музыка не останавливается при смене вкладки

**Решение:**
- Проверьте что используется Web Audio API (не Audio())
- Убедитесь что `audioManager.js` подключен
- Проверьте консоль: должно быть `👁️ Window blur - music paused`

### Проблема: Уведомления о level up не показываются

**Решение:**
- Соберите 50+ яиц для level 2
- Проверьте консоль: должно быть `⚡ Level 2: Опытный`
- Убедитесь что `gamePatch.js` подключен

### Проблема: Рекорд не сохраняется

**Решение:**
- Проверьте localStorage в DevTools (F12 → Application → Local Storage)
- Должен быть ключ `best_score`
- Для VK/Яндекс нужна авторизация

## 🔗 API интеграции

### Яндекс.Игры:

```javascript
// Автоматически вызывается
ysdk.features.LoadingAPI.ready();

// Сохранение рекорда
ysdk.getLeaderboards().setLeaderboardScore('best_score', score);

// Показ рекламы
ysdk.adv.showFullscreenAdv({...});
```

### VK Bridge:

```javascript
// Инициализация
vkBridge.send('VKWebAppInit');

// Сохранение рекорда
vkBridge.send('VKWebAppStorageSet', {
    key: 'best_score',
    value: score.toString()
});

// Показ рекламы
vkBridge.send('VKWebAppShowNativeAds', {
    ad_format: 'interstitial'
});
```
