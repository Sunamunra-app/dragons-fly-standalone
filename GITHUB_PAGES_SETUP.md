2. В разделе **"Branch"** (Ветка):

   - Folder: **/ (root)**

3. Подождите 1-2 минуты

## 🔧 Если что-то не работает

### Проблема: GitHub Pages не включается
**Решение:**
1. Убедитесь что репозиторий публичный (Public)
2. Перейдите в Settings → Pages
3. Выберите main branch и / (root) folder
4. Нажмите Save

### Проблема: Игра не загружается на GitHub Pages
**Решение:**
1. Подождите 2-3 минуты после включения Pages
2. Проверьте консоль браузера (F12) на ошибки
3. Убедитесь что все файлы загружены в репозиторий

### Проблема: 404 ошибка на GitHub Pages
**Решение:**
1. Проверьте что файл `index.html` в корне репозитория
2. Очистите кэш браузера (Ctrl+Shift+R)
3. Подождите 5-10 минут (GitHub Pages иногда обновляется медленно)
